# qsv-medisviewer

QSV MediaViewer(仮称) は、音楽、画像、動画の管理と再生を統合した多機能メディアビューアです。

## ダウンロード

最新版は GitHub の **Releases** ページから入手できます。  

## インストール方法

1. 配布されている zip ファイルを解凍する
2. 解凍したフォルダ内の `QSV MediaViewer.exe` を実行する

**注意:** フォルダ内の `.dll` ファイル等は削除・移動しないでください。アプリケーションが起動しなくなります。

## 動作環境

* **OS:** Windows 10 / 11 (64bit)
